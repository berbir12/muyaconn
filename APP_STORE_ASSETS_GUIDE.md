# App Store Assets Guide

## Required Assets for App Store Submission

### 1. App Icons

#### iOS App Icon
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **File**: `./assets/images/icon.png`
- **Requirements**:
  - No transparency
  - No rounded corners (Apple will add them)
  - High resolution
  - Simple, recognizable design

#### Android App Icon
- **Size**: 512x512 pixels (foreground)
- **Format**: PNG
- **File**: `./assets/images/adaptive-icon.png`
- **Background Color**: #3B82F6 (defined in app.json)

### 2. Splash Screen
- **Size**: 1242x2688 pixels (iPhone X dimensions)
- **Format**: PNG
- **File**: `./assets/images/splash.png`
- **Requirements**:
  - Logo centered
  - Background color: #ffffff
  - No text (logo only)

### 3. App Store Screenshots

#### iPhone Screenshots (Required)
- **6.7" Display (iPhone 15 Pro Max)**: 1290x2796 pixels
- **6.1" Display (iPhone 15 Pro)**: 1179x2556 pixels
- **5.5" Display (iPhone 8 Plus)**: 1242x2208 pixels

#### iPad Screenshots (Optional but Recommended)
- **12.9" Display (iPad Pro)**: 2048x2732 pixels
- **11" Display (iPad Pro)**: 1668x2388 pixels

### 4. App Store Connect Metadata

#### App Information
- **App Name**: SkillHub
- **Subtitle**: Connect with skilled professionals
- **Description**: 
  ```
  Connect with skilled professionals for any task. Post jobs or offer your services on Ethiopia's leading marketplace platform.

  Features:
  • Post jobs and find skilled taskers
  • Offer your services and earn money
  • Real-time messaging and notifications
  • Secure payment processing
  • Rating and review system
  • Location-based matching
  • Professional verification

  Categories: Handyman, Cleaning, Electrical, Plumbing, Painting, Yard Work, and more.

  Perfect for homeowners, businesses, and skilled professionals looking to connect and get work done.
  ```

#### Keywords
- marketplace, services, jobs, taskers, ethiopia, local services, handyman, freelance, cleaning, electrical, plumbing, painting, yard work

#### App Category
- **Primary**: Business
- **Secondary**: Productivity

### 5. App Store Connect Settings

#### Pricing and Availability
- **Price**: Free
- **Availability**: All countries (or specific regions)
- **Age Rating**: 17+ (due to marketplace nature)

#### App Review Information
- **Contact Information**: Your contact details
- **Demo Account**: Test account credentials
- **Notes**: Any special instructions for reviewers

### 6. Privacy Information

#### Data Collection
- **Contact Info**: Name, Email, Phone Number
- **Location**: Precise Location, Coarse Location
- **Identifiers**: User ID, Device ID
- **Usage Data**: App Interactions, Performance Data
- **Diagnostics**: Crash Data, Performance Data
- **Other Data**: Photos, Audio Data, Customer Support

#### Data Usage
- **App Functionality**: Required for core features
- **Analytics**: App performance and usage
- **Developer Communications**: Customer support
- **Advertising**: None (if not using ads)

### 7. App Store Review Guidelines Compliance

#### Content Guidelines
- ✅ No inappropriate content
- ✅ No violence or harmful content
- ✅ No spam or misleading information
- ✅ Professional marketplace platform

#### Technical Guidelines
- ✅ App functions as described
- ✅ No crashes or bugs
- ✅ Proper error handling
- ✅ Good user experience

#### Legal Guidelines
- ✅ Privacy Policy included
- ✅ Terms of Service included
- ✅ Proper permissions usage
- ✅ No copyright violations

### 8. Pre-Submission Checklist

#### App Configuration
- [ ] App name updated in app.json
- [ ] Bundle identifier set (com.skillhub.app)
- [ ] Version number set (1.0.0)
- [ ] Build number set (1)
- [ ] All required permissions added
- [ ] Privacy descriptions added

#### Legal Documents
- [ ] Privacy Policy created and accessible
- [ ] Terms of Service created and accessible
- [ ] Legal pages added to app
- [ ] Contact information updated

#### Assets
- [ ] App icon created (1024x1024)
- [ ] Splash screen created
- [ ] Screenshots prepared
- [ ] All assets optimized

#### Testing
- [ ] App tested on multiple devices
- [ ] All features working properly
- [ ] No crashes or bugs
- [ ] Good performance

### 9. Submission Process

1. **Prepare Assets**: Create all required images
2. **Update App**: Ensure all legal requirements met
3. **Build App**: Create production build
4. **Upload to App Store Connect**: Submit for review
5. **Wait for Review**: Typically 24-48 hours
6. **Address Feedback**: Fix any issues if rejected
7. **Publish**: Release to App Store

### 10. Post-Launch

- **Monitor Reviews**: Respond to user feedback
- **Update Regularly**: Fix bugs and add features
- **Analytics**: Track app performance
- **Marketing**: Promote your app
